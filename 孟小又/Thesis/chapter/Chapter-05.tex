\chapter{总结与展望}

\section{总结}

本文主要针对大规模积云场景的实时渲染这一研究课题进行了深入的研究，通过对比分析近几年关于云场景渲染的国内外研究成果的优缺点，总结并归纳出大规模积云场景实时渲染的重难点，采用基于预计算光照的方法对云粒子进行光照计算、基于相机中心的网格结构对积云场景的分布与体积进行建模，实现了一种针对大规模积云场景的实时渲染算法。本文研究成果如下：

\begin{enumerate}
  \item 针对积云这种参与性介质的场景，光线主要与积云内部的微水滴发生多重散射作用。多重散射的计算量非常大，难以在GPU上做到实时渲染。针对这一难点，采用基于预计算的光照算法，在单位球粒子上对基于路径数值积分的多重散射光照模型和光学深度进行预计算，并将预计算结果存储在三维纹理中，在渲染阶段对纹理采样即可。实验结果表明，该算法能够保证实时渲染帧率的同时，也渲染出了较高的真实感。
  \item 针对大规模积云场景的分布与体积特点，本文采用基于相机中心的嵌套网格结构对积云场景分布进行建模，控制远近不同的层次细节，并以云层的方式对积云体积表达进行建模。实验结果表明，该建模方法能够很好地控制远近不同积云细节，也生成了高真实感的积云体积表达。该建模方法完全基于GPU程序实现，满足实时建模的要求。
  \item 为了积云场景更具真实感，本文在动态建模的基础，不断修改云密度纹理的采样坐标，从而修改积云场景的体素密度数据，实现积云动态生长与消散的效果。
  \item 为了增加整个场景的真实感，本文还实现了简单的天空背景和地形场景的渲染，并将各个场景整合到一起。实验结果表明，本文得到一个较好的户外场景渲染效果。
\end{enumerate}

\section{展望}
本文基于预计算光照的大规模积云场景的实时渲染算法，结合基于相机中心的嵌套网格结构对积云场景的分布和体积表达进行建模，得到了真实感较高的渲染效果以及较高的渲染帧率。但从这两方面来看，本文算法仍存在一定的不足之处，其改进的方向是未来研究工作的重点。未来研究方向主要包含以下几个方面：

\begin{enumerate}
  \item 采用更复杂而且物理精确的光照模型，模拟粒子间的能量交换。
  \item 采用流体动力学Navier-Stokes方程，基于物理地模拟积云场景的动态变化。
  \item 本文算法对于远处的积云场景同样采用构建体素包围盒与视线求交的方法进行渲染，可以采用布告板技术对远处低精度的积云进行渲染，进一步提高本文算法的渲染效率。
  \item 本文可渲染的场景类型还有待增加，如不同云类型的模拟，或者户外场景中其他场景的模拟。
\end{enumerate}
